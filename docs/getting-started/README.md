# Getting Started

Speedwave ships as a single installable application — no Docker Desktop, no Node.js, no Python required.

## Quick Start

1. Download the latest release for your platform from [GitHub Releases](https://github.com/speednet-software/speedwave/releases)
2. Install the application (see [Installation](installation.md) for platform-specific instructions)
3. Launch Speedwave and follow the setup wizard
4. Configure your project (see [Configuration](configuration.md))

## Choose Your Interface

- **Desktop App** — chat UI with project management and system integrations → [Desktop Guide](../guides/desktop.md)
- **CLI** — terminal-based Claude Code via `speedwave` → [CLI Guide](../guides/cli.md)
