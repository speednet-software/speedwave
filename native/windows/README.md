# native/windows — Windows Native OS CLI

Placeholder for the Windows `native-os-cli.exe` Rust binary.

Per [ADR-010](../../docs/adr/ADR-010-mcp-os-as-host-process-per-platform.md), this will be a single Rust binary using `windows-rs` for WinRT API access and `mapi-rs` for Outlook mail/calendar integration.

See [ADR-027](../../docs/adr/ADR-027-native-directory-structure.md) for the directory structure rationale.
