# native/linux — Linux Native OS CLI

Placeholder for the Linux `native-os-cli` Rust binary.

Per [ADR-010](../../docs/adr/ADR-010-mcp-os-as-host-process-per-platform.md), this will be a single Rust binary using `zbus` for D-Bus access and CalDAV (RFC 4791) for calendar integration via GNOME Evolution Data Server.

See [ADR-027](../../docs/adr/ADR-027-native-directory-structure.md) for the directory structure rationale.
